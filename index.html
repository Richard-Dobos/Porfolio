Dusk Game Engine

Dusk Game Engine is a 2D and 3D Game Engine Curently in development

WHY MAKE A GAME ENGINE

There are many game engines out there that have been developed for a long time by huge teams. But none of them are open-source.

The complete engine

Dusk

The final verison of Dusk will not only support game development but it will also be able to create 3D enviroments for showcases. Dusk will also support app development with C# scripting at its heart. Dusk will be compatible with all well known platfoms. In future you will be able to download dusk on GitHub.
