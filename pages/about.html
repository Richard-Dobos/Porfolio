About me

Im 18 years old C++ developer named Richard. Most of the time I develop games or work on Dusk Game Engine and I use C++ with VS 2022 or VS 2019.
I started creating games in Unity when I was 14 years old, at the same time I started learing Python. I even tried machine learning in Python but I ended up giving up on it. Also for 1.5 year I tried using UnrealEngine and for a while I also tried CryEngine. In 2018 I began learing how to code in C++ and also how do game engines work And some graphics APIs like OpenGl, Vulkan and DirectX. And at the end of 2020 I tried to write my first 2D game engine. Dusk began as my second try to make 3D game engine. But now its more of an idea that there are no big open-source game engines.
